# jacob.rath.website

Static GitHub Pages version of [jacobrath.com](https://www.jacobrath.com/).

## Deploy

In GitHub, open **Settings → Pages**, choose **Deploy from a branch**, select `main`, and use the `/ (root)` folder.

The Squarespace portfolio used a server-side password. GitHub Pages is static, so this version does not include a password gate by default; any client-side gate would be discoverable in the source. The WashScore privacy and support pages are included as standalone pages.
